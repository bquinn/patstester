Send Order with Campaign

Helper page that lets you create a campaign and immediately send up to two orders on that campaign, all with one form. It also inserts campaign dates, publisher ID/name and campaign ID so you won't have to edit the order payload(s).

Request order revision

Request a seller to revise an order, whether or not they have sent a revision. Access via link from orders/revisions list and order revision detail page.

Return order revision

If you don't like an order revision you have received, you can "return" it to the seller. Linked from order revisions list and order revision detail page.
